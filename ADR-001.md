# ADR-001: Linguagem de implementação

**Status:** Aceito
**Data:** 28/09/2025

## Contexto

Precisamos escolher a linguagem principal para implementar o sistema de reconhecimento com câmeras (detecção, identificação, notificação e dashboard).

## Decisão

Optamos por **Python**.

## Consequências

- **Positivas:**

  - Grande ecossistema para visão computacional (OpenCV, DeepFace, YOLOv8, Mediapipe).
  - Comunidade ativa e documentação rica.
  - Rápida prototipagem.

- **Negativas:**

  - Menor performance bruta comparada a C++/Rust.
  - Dependência de GPU/CPU otimizada para manter tempo real.
